What is SkillSet
=====================

SkillSet is a tool designed to improve the efficiency and usability of layout design in Cadence Virtuoso.
It offers a range of practical features tailored for layout engineers and will be continuously updated with new functionalities.

Developed in the Skill programming language (LISP), SkillSet was built with insights and inspiration from the `Cadence Community <https://community.cadence.com>`_
