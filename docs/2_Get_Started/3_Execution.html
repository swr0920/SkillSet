How to Launch the SkillSet GUI¶

Using the SkillSet GUI menu is very simple. Just open any layout cellview (including LayoutL, LayoutXL, or LayoutEAD) that you want to work on. You will find the SkillSet menu in the Cadence Layout Windows Menu bar.
